# Продуктовый контекст

## Проблема пользователей
Многие люди хотят получать профессиональные рекомендации по уходу за кожей, но:
- Не могут позволить себе регулярные визиты к косметологу
- Не знают, какие средства подходят именно их типу кожи
- Теряют информацию о предыдущих рекомендациях
- Хотят быстрые ответы в любое время

## Решение
ИИ-ассистент "Мой личный косметолог" предоставляет:
1. **Мгновенные консультации** - ответы на вопросы 24/7
2. **Персонализацию** - учет типа кожи, возраста, проблем, аллергий
3. **Анализ фото** - визуальная оценка состояния кожи
4. **Историю** - сохранение всех рекомендаций и анализов
5. **Безопасность** - четкие границы: нет медицинских диагнозов, только уход

## Пользовательские сценарии

### Сценарий 1: Первый визит
1. Пользователь попадает на лендинг
2. Регистрируется через email или OAuth (Google/GitHub)
3. Заполняет профиль (возраст, тип кожи, проблемы)
4. Принимает медицинский дисклеймер
5. Начинает чат с ИИ-ассистентом

### Сценарий 2: Консультация
1. Пользователь заходит в раздел "ИИ-Консультация"
2. Задает вопрос о конкретной проблеме (например, "как ухаживать за комбинированной кожей?")
3. Получает персонализированную рекомендацию с учетом его профиля
4. История сохраняется для будущего доступа

### Сценарий 3: Анализ фото
1. Пользователь загружает фото лица
2. Gemini Vision анализирует состояние кожи
3. Система предоставляет рекомендации на основе анализа
4. Фото и анализ сохраняются в истории

### Сценарий 4: Просмотр истории
1. Пользователь открывает предыдущие консультации
2. Может продолжить старый диалог или начать новый
3. Просматривает предыдущие фотоанализы

## Ключевые метрики успеха
- Регистрация и заполнение профиля
- Частота использования чата
- Количество загруженных фото для анализа
- Возврат пользователей

## Принятые решения
- **Бесплатная модель** (на текущем этапе) - без ограничений на использование
- **Принудительный дисклеймер** - перед первым использованием чата/анализа
- **Сохранение контекста** - ИИ помнит предыдущие сообщения в рамках сессии
- **Безопасность данных** - фото и данные профиля принадлежат пользователю
