# Мой личный косметолог (Beautify)

## Цель проекта
Создание веб-приложения с ИИ-ассистентом для персонализированных рекомендаций по уходу за кожей лица.

## Основные задачи
- Предоставить пользователям доступ к ИИ-консультациям по косметологии
- Анализ фото кожи с помощью компьютерного зрения (Gemini Vision)
- Сохранение истории консультаций и анализов
- Управление профилем пользователя с данными о типе кожи и проблемах

## Рамки проекта
### Включено
- Регистрация и аутентификация пользователей (Supabase Auth)
- ИИ-консультация через чат с контекстом
- Анализ фото кожи с сохранением результатов
- История консультаций и фотоанализов
- Управление профилем (возраст, тип кожи, проблемы, аллергии, цели)
- Поддержка темной/светлой темы
- Медицинский дисклеймер

### Исключено (на текущем этапе)
- Оплата и подписки
- Интеграция с реальными косметологами
- Мобильное приложение
- Push-уведомления

## Целевая аудитория
Пользователи, интересующиеся домашним уходом за кожей, которым нужны персонализированные рекомендации без посещения косметолога.

## Ключевые ограничения
- ИИ не ставит медицинские диагнозы
- Требуется принятие дисклеймера перед использованием
- Фото пользователей хранятся в зашифрованном виде
- Rate limiting на API endpoints

## Статус проекта
Активная разработка (MVP). Основной функционал реализован, работают чат, анализ фото, профиль, история.
