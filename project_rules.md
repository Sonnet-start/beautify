# Правила проекта

- **Язык общения**: Русский.
- **Пакетный менеджер**: Bun.
- **Линтинг и форматирование**: Biome. Автоматически исправлять ошибки после внесения изменений.
- **Терминал**: PowerShell (Windows).
- **Стек Frontend/Metaframework**: Next.js (latest, App Router), React, TypeScript, Tailwind CSS, Shadcn UI, Framer Motion.
- **Стек Backend**: Supabase (Auth, Database, Storage, Edge Functions). Next.js API Routes (при необходимости).
- **Сервер разработки**: Управляется пользователем. Не запускать, не останавливать, не проверять.
- **Просмотр файлов**: Не спрашивать разрешения, использовать инструменты сразу.
- **Выбор решений**: Автоматически выбирать лучшее решение на основе мировых практик, не предлагать варианты.
- **Изменения**: Только запланированные или запрошенные. Дополнительные предложения — в конце отчета.
- **AI Провайдер**: Google AI (Gemini 3 Flash via OAuth) вместо OpenAI.
- **Управление состоянием**: Zustand (UI) + TanStack Query (Data).
- **Стили**: Дизайн "Wow"-эффекта, премиум, стеклянный морфизм (Glassmorphism). Шрифт: Стильный и строгий (например, Manrope или Playfair Display — не Inter).
- **Браузер**: Не открывать дополнительные вкладки.
- **Git**: Не коммитить автоматически. Только по команде.
