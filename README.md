# 🌸 Мой личный косметолог

Веб-приложение для персонализированных рекомендаций по уходу за кожей лица с использованием AI-технологий.

## 📋 Описание проекта

«Мой личный косметолог» помогает пользователям получать профессиональные рекомендации по уходу за кожей на основе индивидуальных потребностей.

## ✨ Основные возможности

- 🤖 AI-рекомендации на базе Gemini 3 Flash (OAuth)
- 📸 Анализ фото кожи
- 💬 Чат с AI-косметологом
- 📅 Календарь ухода
- 📊 Отслеживание прогресса
- 📱 Адаптивный дизайн

## 📁 Структура проекта

```
├── app/                 # Next.js App Router
├── components/          # UI компоненты
├── lib/                 # утилиты и сервисы
├── public/              # статические ресурсы

```

## 🛠 Технологический стек

### Frontend
- Next.js 16 (App Router) + React 19 + TypeScript
- Tailwind CSS v4 + shadcn/ui
- Zustand для UI состояния
- TanStack Query для сетевых запросов
- Framer Motion для анимаций

### Backend и инфраструктура
- Supabase (Auth, Database, Storage)
- Next.js API Routes (при необходимости)

### AI
- Google Gemini 3 Flash via OAuth (`ai`, `ai-sdk-provider-gemini-cli`)

### Инструменты
- Biome для lint/format
- Bun как пакетный менеджер

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
bun install
```

### 2. Настройка окружения
Создайте `.env.local` и добавьте:
```bash
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
```

### 3. Авторизация в Gemini (OAuth)
```bash
gemini login
```

### 4. Запуск сервера разработки
```bash
bun run dev
```
